    Comenzar a trabajar con NI-DAQmx

    Actualizado 28 oct 2025

    Información general

    Esta página está destinada a ayudarle a aprender los fundamentos de la programación con NI-DAQ™mx. Le guiaremos desde la verificación del funcionamiento de su dispositivo en Measurement & Automation Explorer (MAX) hasta la programación de aplicaciones de adquisición de datos utilizando LabVIEW. Está dirigido tanto para principiantes que desean aprender cómo usar el DAQ Assistant, así como usuarios experimentados que desean provechar la funcionalidad avanzada de NI-DAQmx. Para un recorrido paso a paso, consulte la trayectoria de aprendizaje Adquisición de datos usando NI-DAQmx y LabVIEW.

    Descargar NI-DAQmx

    • Descargar NI-DAQmx

    Instalar su hardware

    • Disponemos de diferentes tutoriales para comenzar según la combinación de hardware y software que vaya a utilizar. Encuentre el tutorial adecuado para usted.
    • Consideraciones sobre el cableado de campo y el ruido

    Utilice NI MAX para probar y configurar su hardware

    NI Measurement & Automation Explorer (MAX) proporciona acceso a su hardware NI. Es un software gratuito que no puede descargarse por separado, pero se incluye y se instala automáticamente con todos los controladores NI y NI System Configuration.

    Ayuda para comenzar con DAQ Assistant

    El DAQ Assistant es una interfaz gráfica fácil de usar para configurar tareas y canales de medición, y para personalizar la temporización, disparos y escalas sin necesidad de programar.

    Programación en LabVIEW con funciones NI-DAQmx

    Si está recopilando datos de forma continua desde su dispositivo DAQ, las funciones NI-DAQmx le permiten configurar canales, tareas, adquirir y registrar puntos de datos.
